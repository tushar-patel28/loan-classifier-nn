A neural network built with TensorFlow/Keras to predict loan approval decisions.

## Features
- Compares 5 different neural network architectures
- Uses StandardScaler for feature normalization
- Implements early stopping to prevent overfitting
- Outputs accuracy, precision, recall, and F1-score metrics

## Requirements
```
pip install pandas numpy scikit-learn tensorflow openpyxl
```

## Usage
Run the Jupyter notebook `loan_classifier_nn.ipynb`
